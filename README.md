# Prime Agro Farm

A modern agricultural website for Prime Agro Farm, Uganda's leading dairy and livestock breeding facility.

## Features

- **Dairy Farming**: Automated milking and quality control.
- **Silage & Hay**: Premium forage production.
- **Training Workshops**: Agricultural education and management.
- **Cattle Breeding**: Superior genetics for dairy and beef.
- **Admin Dashboard**: Manage news, gallery, and leadership.
- **Real-time News**: Integrated with Supabase.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Database**: Supabase
- **Styling**: Tailwind CSS & Framer Motion
- **Icons**: Lucide React
- **Email**: EmailJS & Resend

## Setup

1. Clone the repo
2. Install dependencies: `bun install`
3. Set up environment variables in `.env.local`
4. Run development server: `bun dev`
